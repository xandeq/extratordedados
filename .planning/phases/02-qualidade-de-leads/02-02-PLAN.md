    **normalize_phone_br(raw: str) -> dict:**
    - Strip to digits, prepend +55 (handle leading 55 and leading 0)
    - `phonenumbers.parse(parse_str, 'BR')` + `is_valid_number()` check
    - `PhoneNumberFormat.E164` → e164 string, `PhoneNumberFormat.NATIONAL` → national string
    - `phonenumbers.number_type()` → type: 'mobile' if MOBILE, 'landline' if FIXED_LINE or FIXED_LINE_OR_MOBILE, 'toll_free' if TOLL_FREE, else 'unknown'
    - whatsapp_id: only for MOBILE type → `e164.lstrip('+') + '@c.us'`
    - Per Pitfall 5: if type is FIXED_LINE_OR_MOBILE and national number string has 8 digits → treat as 'landline', whatsapp_id=None

    **compute_lead_quality_score(lead: dict) -> dict:**
    - Email (30 pts): no email=0, invalid=0, free_provider=15, corporate_valid=30
    - Phone (20 pts): no phone=0, invalid=0, mobile=20, landline=12, other=5
    - Completeness (20 pts): sum of field weights: company_name=4, email=3, phone=3, website=2, city=2, state=2, cnpj=2, category=1, address=1 (cap at 20)
    - Freshness (15 pts): use `last_verified_at` or `captured_at` or `extracted_at`; days<=60→100, days<=180→70, days<=365→40, else→10; unknown→50; freshness_pts = int(freshness * 0.15)
    - CNPJ enrichment (10 pts): cnpj_enriched=True→10, has cnpj→5, else→0
    - Source quality (5 pts): map={'google_maps':5,'directories':4,'search_engines':3,'api_enrichment':4,'imported':2,'website_crawl':3}, default=2
    - Total 0-100, grade: >=80→'A', >=60→'B', >=40→'C', >=20→'D', else→'F'
    - tier: 'premium' if A/B, 'medio' if C, 'basico' if D/F
    - Return: `{score, grade, tier, freshness, breakdown: {email, phone, completeness, freshness, cnpj, source}}`
