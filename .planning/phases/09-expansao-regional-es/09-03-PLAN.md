**Step 2 — Add useEffect to fetch regions:**
Add a separate useEffect (do not merge with the existing pipeline-config fetch):
```typescript
useEffect(() => {
  setRegionsLoading(true)
  api.get('/api/admin/regions')
    .then(res => setRegions(res.data.regions || []))
    .catch(() => setRegions([]))  // graceful: 403 on non-admin shows empty list
    .finally(() => setRegionsLoading(false))
}, [])
```

**Step 3 — Add helper function for badge color:**
Add this helper near the top of the component (after state declarations):
```typescript
const isRecentlyUsed = (lastUsedAt: string | null): boolean => {
  if (!lastUsedAt) return false
  const sevenDaysAgo = new Date(Date.now() - 7 * 24 * 60 * 60 * 1000)
  return new Date(lastUsedAt) > sevenDaysAgo
}
```
