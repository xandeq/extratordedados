```python
def _get_outscraper_key():
    """Fetch Outscraper API key from AWS SM (with env fallback)."""
    key = os.environ.get('OUTSCRAPER_API_KEY')
    if not key:
        try:
            import boto3
            sm = boto3.client('secretsmanager', region_name='us-east-1')
            secret = sm.get_secret_value(SecretId='tools/outscraper')
            key = json.loads(secret['SecretString']).get('OUTSCRAPER_API_KEY')
        except Exception as e:
            print(f"[outscraper] AWS SM error: {e}")
    if not key:
        raise ConfigError("[OUTSCRAPER] API key n├úo dispon├şvel ÔÇö crie conta em outscraper.com e salve em AWS SM tools/outscraper")
    return key
```

**Done:** `_get_outscraper_key()` exists in app.py. `outscraper` package in requirements.txt. AWS SM secret `tools/outscraper` created (empty key placeholder).

### Task 2 ÔÇö `process_outscraper_massive()` thread function

**Files:**
- `app/backend/app.py` (modify)

**Action:**

Add `process_outscraper_massive(batch_id, jobs_data, user_id)` after `process_local_business_data_massive()` in app.py (after line ~11550). Use `@_persist_thread_errors('outscraper')` decorator. Follow the same structure as `process_local_business_data_massive()` exactly ÔÇö same try/except hierarchy, same `quota_exceeded` flag pattern, same `_save_leads_to_batch()` call signature, same scraper_log calls.

Key differences from local_business_data:
- Uses `outscraper` Python client: `from outscraper import ApiClient` (lazy import inside function)
- `client.google_maps_search([query], limit=20, language="pt", region="BR", fields=["name","phone","email","full_address","site","category"])` ÔÇö synchronous call (not async_request)
- Result structure: `results[0]` is the list of businesses for the single query
- Lead mapping from Outscraper response fields: `name` Ôćĺ `company_name`, `email` Ôćĺ `email`, `phone` Ôćĺ `phone`, `site` Ôćĺ `website`, `full_address` Ôćĺ `address`, use `niche` for `category`
- Source: `'outscraper_maps'`
- Quota detection: catch HTTP 429 or any exception where `'429' in str(err).lower() or 'quota' in str(err).lower() or 'limit' in str(err).lower()` Ôćĺ set `quota_exceeded = True`
- ConfigError from `_get_outscraper_key()` Ôćĺ also sets `quota_exceeded = True` (no key = skip all jobs)
- Get key once before the loop, not per-job

Lambda closure pattern for `_massive_retry` (CRITICAL ÔÇö avoid late binding):
```python
result, err = _massive_retry(
    lambda q=query: client.google_maps_search(
        [q], limit=20, language="pt", region="BR",
        fields=["name", "phone", "email", "full_address", "site", "category"]
    ),
    provider='outscraper',
    query=query
)
```
